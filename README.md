# Gulp Project Boiler Plate

## About
This is the starting point that I use when I build a new project.

## Technologies
* NPM
* Gulp
* CoffeeScript
* Stylus
* Jade

## Getting Started
1. Install gulp dependencies
	
	```
	npm install
	```
2. Update package.json
	* Update your Name, Description, Repository, Homepage etc.
3. Run Gulp
	
	``` 
	gulp
	```
